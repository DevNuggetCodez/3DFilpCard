# 🎴 3D Flip Card

A simple and interactive **3D Flip Card** built using HTML and CSS. This project demonstrates how to use CSS transforms, perspective, and hover effects to create smooth UI animations.

---

## 📌 Features

- 3D flip animation on hover  
- Front and back card design  
- Smooth transition effects  
- Clean and beginner-friendly code  
- Fully responsive structure (can be extended)
---

## 🛠️ Technologies Used

- HTML5  
- CSS
